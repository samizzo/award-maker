Help

This tool generates responsive award laurel layouts using UIkit 2. There are three layouts generated:

  • Large - three awards per row
  • Medium - two awards per row
  • Small - one award per row

The UIkit CSS switches between these layouts automatically based on the window resolution of the browser. The generated HTML is mostly self-contained; the only two external references it contains are to the Raleway font and the UIkit 2 CSS. If you are already using these you can remove them.

To use the generated HTML in your own web pages, copy it to the clipboard or save it to disk with the buttons in the HTML preview window. Then copy the CSS into your own stylesheet, and copy the HTML between the body tags into your page. If you wish to switch fonts, simply change the font-family in the generated CSS.

There is also a presskit snippet generated for copying into your presskit().

Finally, the JSON snippet can be kept in source control as your master award definition file. You can modify the JSON and paste into the text area to automatically update the awards.

TL;DR usage:

  1. Add awards to the table.
  2. Save the HTML code or copy to clipboard.
  3. Insert into your own webpage.
  4. Profit!
