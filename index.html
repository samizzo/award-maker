Help

This tool generates responsive award laurel layouts using UIkit 2. There are three layouts generated:

  • Large - three awards per row
  • Medium - two awards per row
  • Small - one award per row

To use the generated HTML in your own web pages, copy it to the clipboard or save it to disk with the buttons in the HTML preview window. Then copy the CSS into your own stylesheet, and copy the HTML between the body tags into your page. If you wish to use a different font, simply change the font-family in the generated CSS.

You can remove the UIkit 2 and font imports if you are already using UIkit 2 or don't wish to use the default font.

There is also a presskit snippet generated for copying into your own presskit().

Finally, the JSON snippet can be kept in source control as your master award definition file. You can modify the JSON and paste into the text area to automatically update the awards.

TL;DR usage:

  1. Add awards to the table.
  2. Save the HTML code to disk or copy it to the clipboard with the buttons.
  3. Insert it into your own webpage.
  4. Profit!
